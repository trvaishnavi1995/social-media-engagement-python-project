# social-media-engagement-python-project
Social Media Engagement Analytics project using Python, Pandas, NumPy, Matplotlib, Seaborn, and Plotly.
# Social Media Engagement Analytics Using Python

## 📌 Project Overview
This project analyzes social media engagement data using Python.

The analysis includes:
- Data Cleaning
- Exploratory Data Analysis
- Statistical Analysis
- Data Visualization
- Business Insights

---

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly

---

## 📊 Key Insights
- Video/Reel content generated higher engagement
- Positive sentiment posts performed better
- Verified accounts showed stronger interaction
- Posting time affected impressions significantly

---

## 📈 Visualizations Included
- Heatmap
- Bar Charts
- Scatter Plot
- Line Plot
- Histogram
- Pie Chart
- Interactive Plotly Charts

---

## 🚀 Project Outcome
This project demonstrates how Python can be used for:
- Social Media Analytics
- Business Intelligence
- Data Visualization
- Data-driven Decision Making
